PlayMusic 플러그인

플러그인 소개

PlayMusic은 마인크래프트 서버에서 URL 기반으로 음악을 재생할 수 있도록 설계된 기초 플러그인입니다.
해당 플러그인을 수정하여 서버 내에서 높은 퀄리티의 사운드, 음악들을 재생할 수 있게 제작할 수 있습니다.

주요 기능

  • URL 기반 음악 재생 (YouTube 등 지원).
  • 특정 플레이어 또는 전체 서버에 음악 출력.
  • 명령어를 통해 음악 재생 및 중단 가능.

설치 방법

  1. 다운로드한 기초 플러그인을 서버에 맞게 수정합니다.
  2. 수정한 JAR 파일을 서버의 /plugins 폴더에 복사합니다.
  3. 서버를 재시작하여 플러그인을 활성화합니다.

사용 방법

1. 음악 재생

다음 명령어를 사용하여 URL 기반으로 음악을 재생할 수 있습니다:

/play [URL]

예시:

 /play https://www.youtube.com/watch?v=dQw4w9WgXcQ

2. 음악 중단

현재 재생 중인 음악을 중단하려면 다음 명령어를 사용하세요:

/stop

주의사항

  • YouTube 또는 기타 음원 사이트의 콘텐츠를 사용할 때 저작권 규정을 준수해야 합니다.
  • 플러그인은 Node.js 서버와 함께 사용되며, Node.js 설정이 필요합니다.
  • 플러그인 오류 및 오사용으로 인한 오류는 책임지지 않습니다.

라이센스

기초 플러그인으로 누구나 수정, 재배포가 가능합니다.
